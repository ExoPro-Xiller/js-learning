Build something that feels a little like a tiny game/account system. Simple enough for your level, but still fun.

🎮 Mini Account System

You have one user account with:

name
coins
level
logged-in status
🌟 Your Object
let user = {
  name: "Ahmad",
  coins: 100,
  level: 1,
  isLoggedIn: false
};
✅ Functions To Make
1. login()

If not logged in:

set isLoggedIn = true
print message
2. logout()

If logged in:

set isLoggedIn = false
3. earnCoins(amount)

Adds coins.

Example:

earnCoins(50)
4. buySword()

Sword costs 80 coins.

If enough coins:

subtract coins
print success

Else:

print “Not enough coins”
5. levelUp()

Increase level by 1.

6. showUser()

Print:

Name: Ahmad
Coins: 150
Level: 2
Logged In: true
🖱 Add Simple Buttons

Example:

<button onclick="earnCoins(50)">Earn Coins</button>

You can make buttons for:

login
logout
earn coins
buy sword
level up
show user

objects
